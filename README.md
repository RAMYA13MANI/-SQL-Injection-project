# Exploiting SQL Injection Vulnerability using SQLMap, Havij and JSQL

## Project Overview
This project demonstrates the identification and exploitation of SQL Injection vulnerabilities in vulnerable web applications. The goal of the project is to show practical understanding of web application penetration testing and database exploitation techniques.

## Objectives
- Identify SQL Injection vulnerabilities in a web application
- Exploit the vulnerability to access the database
- Extract database information such as tables and records
- Demonstrate real-world penetration testing methodology

## Tools Used
- SQLMap  
- Havij  
- JSQL  

## Methodology
1. A vulnerable web application was selected for testing.
2. Input parameters were analyzed to detect SQL Injection points.
3. SQLMap was used to enumerate databases available on the server.
4. Tables and columns were identified and extracted.
5. Sensitive data was retrieved to demonstrate the impact of the vulnerability.

- Web Application Penetration Testing

The project demonstrates practical knowledge of SQL Injection and highlights the importance of securing web applications against common vulnerabilities.
